# DESKER Design Guide

## Brand Tone
- 미니멀하고 정돈된 인상
- 정보 전달 중심
- 과한 장식 금지
- 화려한 마케팅형 UI보다 조용한 프리미엄 상품 진열형 UI 우선

## Core Visual Principles
- 넓은 여백 사용
- 카드 테두리보다 배치와 간격으로 구분
- 단정한 그리드
- 제품 이미지 중심
- 텍스트는 검정/짙은 회색 위주
- 배경은 화이트 또는 매우 옅은 그레이

## Layout
- 고객용: 모바일 우선, 데스크톱 확장형
- 관리자용: 데스크톱 우선
- 상단 헤더는 심플한 수평 구조
- 활성 메뉴는 underline 중심

## Colors
- Background: #FFFFFF
- Sub Background: #F5F5F5
- Border: #E5E5E5
- Primary Text: #111111
- Secondary Text: #666666
- Muted Text: #888888

## Product List UI
- 상품 카드는 box 느낌보다 여백 중심
- 제품 이미지가 가장 먼저 보이도록
- 제품명, 규격, 가격은 정돈된 정렬 유지
- hover는 매우 약하게

## Search
- 검색창은 단정하고 평평한 스타일
- 과한 radius 금지
- 연한 배경과 얇은 경계 사용

## Tabs
- 구역 탭은 심플한 텍스트형 또는 underline형
- 현재 선택된 탭만 명확하게 강조
- 과도한 pill 디자인 금지

## Buttons
- 주 버튼은 블랙 또는 아주 짙은 톤
- 보조 버튼은 화이트 배경 + 얇은 보더
- 과한 그림자 금지

## Forbidden
- 강한 gradient 금지
- 과한 shadow 금지
- 네온/강한 포인트 컬러 금지
- 과도한 animation 금지
