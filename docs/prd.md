# 디지털 가격 POP PRD

## 서비스 목적
가구 매장에서 기존 지류 가격표를 제거하고, QR코드를 통해 고객이 모바일/PC에서 상품 정보를 조회하고 견적을 담아 직원에게 전달할 수 있는 디지털 가격 POP 서비스를 구축한다.

## 고객 핵심 흐름
1. 고객이 특정 매장/존/구역의 QR을 스캔한다.
2. 해당 구역 상품 목록이 기본 노출된다.
3. 고객은 상단 탭으로 같은 매장 내 다른 구역을 이동 조회할 수 있다.
4. 고객은 검색창에서 제품명 또는 제품코드로 현재 매장 전체 상품을 검색할 수 있다.
5. 고객은 상품 상세에서 옵션을 선택하고 견적서에 담는다.
6. 고객은 여러 구역/QR을 이동해도 같은 견적서를 유지한다.
7. 고객은 이름/전화번호를 입력하여 견적을 직원에게 전달한다.

## 고객 검색 정책
- 검색 기준: 제품명, 제품코드
- 검색 범위: 현재 접속한 매장 전체 상품
- 검색어 입력 시 매장 전체 검색 결과 노출
- 검색어 삭제 시 현재 선택된 구역 탭 상품 목록으로 복귀

## 구역 탭 정책
- 기본 탭은 QR로 진입한 현재 구역
- 같은 매장 내 다른 구역을 탭으로 이동 조회 가능
- 탭 이동 시 견적서는 유지됨

## 견적 정책
- 다른 QR 또는 다른 구역 탭으로 이동해도 견적 유지
- 견적은 당일 자정까지 유지
- 견적 초기화 버튼 제공

## 관리자 구조
- 메인관리자: 전체 매장/상품/QR/로그 관리
- 매장관리자: 본인 매장 구역별 상품 편성 및 고객 견적 확인
